Installation #back to top

  • Step 1 - Unzip the HIGH - Project Management System.zip which you have downloaded from the Envato Market.
  • Step 2 - Find the Code folder from the downloaded package and take out all the data from Code folder and upload to your server. Make sure to copy .htaccess file.
  • NOTE : - Please chmod application/config/database.php, install/assets and all files inside this folder to 777.
  • Step 3 - Open your web browser and go to your domain url. For example: http://yourdomain.com Now you will be redirected to the installation page.
  • Step 4 - Enter your the Database Details, if not created databse then create a MySql Database via your web hosting providers Control Panel. Then, enter your Admin Email and Password which will be required to login in the application. This credentials will needed on everytime you login.
  • NOTE : - Password requires 8 characters minimum and no whitespace.
  • Step 5 - Now click on install and wait untill process finished. The system will create necessary tables in database, files and configuration, Then you will be automatically redirected to the login page.
  • NOTE : - If you get 404 OR 505 error then wait for 30 seconds and refresh the page. Again make sure .htaccess is uploaded.
  • NOTE : - HIGH is now ready to run but we recommended you to first setup and customized HIGH application according to your needs from settings section.
  • You can change admin dashboard color and CSS stuff from application/assets/css/custom.css, style.css, and components.css file.

Projects #back to top

  • All you project will be listed here with some basic information. You can manage your projects here like create, update, delete, add users and filter projects.
  • When you click on one specific project you will get all the information about this project, this page known as project details page. Here also you can manage your project like create, update, delete and some extra features.

Tasks #back to top

  • All you tasks will be listed here with some basic information. You can manage your tasks here like create, update, delete assing them to any user. You and user can drag and drop task in the process queue. From projects section you can select tasks for specific project else all task will be listed here.

Users #back to top

  • All system user will be listed here with some basic profile statistics. Using this application you can manage your team members easily. Here you can Add, View, Edit, Delete, Active and Deactive users from the application.

Permissions #back to top

  • NOTE : - Do not give settings permissions to the users.
